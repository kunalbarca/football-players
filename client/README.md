# Best Football Players
## _A autocomplete frontend service_

- Search best football players
- With autocomplete
- ✨Magic ✨

This service is dockerized with its backend service.

## Installation

```sh
docker-compose build
docker-compose up
docker-compose stop

To run the frontend - 
http://localhost:3000/
To run the backend - 
http://localhost:9000/api/v1/footballers
```
